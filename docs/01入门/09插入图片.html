插入图片

插入图片

步骤

  1. 插入
  2. 图片
  3. 弹框选择图片，选好图片后，就插入成功了，选项卡自动从“插入”切换到格式
  4. 点击压缩图片（为了节省PPT占用的存储空间）
  5. 在弹出的压缩图片操作界面取消勾选压缩选项中的 仅应用于此图片，这样整个PPT文件中的图片都会被压缩
  6. 目标输出中选择屏幕(150 ppi)：适用于网页和投影仪(S)，这样的压缩大小比较适合演示

裁剪

选中图片 --> 格式 --> 裁剪

裁剪为16：9的图片，使其可以铺满画布
